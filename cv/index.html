Research Experience

  • An Intelligent Drone Testbed for Control Systems and Verification,

    Developed a custom onboard software for the drone to autonomously control the drone’s motion and operations.
    M.Sc. Thesis, Supervisor: Dr. P. Ferguson

  • Intelligent vibration control with self-sensing piezoelectric actuator,
    Isfahan University of Technology, 2016 - 2018

  • Auto Code Generation for Onboard Space Object Detection and Flight Software Applications,

  • ManitobaSat Satellite’s Onboard Computer and Flight Software Leader,
    University of Manitoba, 2018 - 2021
    Designed a modular onboard computer (OBC) for ManitobaSat-1, a 3U sized CubeSat satellite. The OBC was a system on a chip that used an MRAM.
